## 🟡 Phase 01: JavaScript Fundamentals

> **Goal:** Build rock-solid understanding of JavaScript before applying it to automation.

### Topics Covered

- ✅ **Variables & Data Types** — `var`, `let`, `const`, primitives, objects
- ✅ **Identifier Rules** — Valid naming, reserved words, case-sensitivity
- ✅ **Naming Conventions** — camelCase, PascalCase, UPPER_SNAKE_CASE, Hungarian notation
- ✅ **Comments** — Single-line, multi-line, JSDoc, best practices
- 🔄 **Control Flow** — `if/else`, `switch`, loops (`for`, `while`, `do-while`)
- 🔄 **Arrays & Objects** — Creation, manipulation, iteration, destructuring
- 🔄 **Functions** — Declarations, expressions, arrow functions, closures, callbacks
- 🔄 **ES6+ Features** — Template literals, spread/rest, destructuring, modules
- 🔄 **OOP in JS** — Prototypes, classes, inheritance, encapsulation
- 🔄 **Async JavaScript** — Callbacks, Promises, `async/await`, event loop

### How to Run JS Files

```bash
# Navigate to the chapter folder
cd chapter_02_JavaScript_Concepts

# Run any file with Node.js
node 06_Identifier_Rules.js
node 07_Naming_Conventions.js
node 08_Comments.js
```

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [VS Code](https://code.visualstudio.com/) (recommended editor)
- Git

### Setup

```bash
# Clone the repository
git clone <repository-url>
cd learnPlayWright2x

# Install dependencies (when Playwright is added)
npm install

# Run JavaScript examples
node chapter_02_JavaScript_Concepts/06_Identifier_Rules.js

# Run Playwright tests (when added)
npx playwright test
```
