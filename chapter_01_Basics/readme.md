# Chapter 01: Basics

This chapter covers the absolute fundamentals of JavaScript and the Node.js environment. It is the starting point for running your first scripts and verifying your setup.

---

## Topics Covered

### 1. First JavaScript Program
- Running a simple script using `console.log()`
- Printing messages to the terminal

### 2. Variables and Logging
- Declaring variables using `let`
- Assigning values to variables
- Printing variable values to the console

### 3. Verifying the Environment
- Using `process.platform` to check the operating system (darwin, win32, linux)
- Using `process.arch` to check the system architecture (x64, arm64)
- Using `process.version` to check the installed Node.js version

### 4. Functions and Loops (Hot Code)
- Defining a basic function with parameters and a return value
- Using a `for` loop for repeated execution
- Understanding how JavaScript handles intensive tasks and function calls

---

## Key Takeaways
- `console.log()` is the primary way to output information in JavaScript.
- `let` is used to declare variables that can be reassigned later.
- The `process` object provides useful information about the Node.js runtime environment.
- Functions are reusable blocks of code, and loops allow you to execute code multiple times.
